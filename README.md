Docker
=========

Install docker with docker-compose for a number of linux ditros.

Requirements
============
- Ansible 2.3.0.0
- Change your private_key_file path
- change-your-remote-user
- change-your-ip-here

Usage
-----

``` sh
 ansible-playbook docker.yml
```

Alternatively you can install maven and java jdk 8
